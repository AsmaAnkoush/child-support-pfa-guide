# Image Placeholders Guide

Replace the following placeholder images by adding real images to this folder with the EXACT same filenames:

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| placeholder-logo.png | Site logo | 120x120px (square) |
| placeholder-hero.jpg | Hero section main image (facilitator with children) | 800x800px (square) |
| placeholder-children.png | Small children image in info bar | 280x280px |
| placeholder-cce-logo.png | Center for Continuing Education logo | 360x120px |
| placeholder-unicef-logo.png | UNICEF logo | 360x120px |

After adding the real images with these exact names, the placeholders in the HTML will automatically display the new images.
